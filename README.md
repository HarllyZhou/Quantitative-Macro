# Quantitative-Macro

## This is Harlly's folder for ECON 5345 Quantitative Macroeconomics. It contains the slides, assignments, codes, papers, and research agenda.

## R setup (recommended)

Open `Quantitative-Macro.Rproj` in RStudio. For a reproducible package environment, run:

```bash
Rscript scripts/r_setup.R
```

Then restart R (or reopen the project) and run `renv::snapshot()`.
